# تنفيذ المخزن الدائري - Circular Buffer Implementation

## وصف المشروع
هذا المشروع هو تطبيق عملي بلغةC** لمفهوم **المخزن الدائري (Circular Buffer)**. تم تصميمه لإدارة البيانات في مساحة ذاكرة ثابتة باستخدام مؤشرات الرأس (Head) والذيل (Tail).

## الميزات البرمجية
* **المنطق الدائري:** استخدام عملية باقي القسمة `(index + 1) % SIZE` لضمان دوران المؤشرات داخل المصفوفة.
* **معالجة الأخطاء:** الكود يتضمن شروطاً للتعامل مع حالات الامتلاء (**Overflow**) وحالات الفراغ (**Underflow**).
* **معالجة النصوص:** البرنامج يقوم باستقبال اسم المستخدم، إضافة اللاحقة المطلوبة (" CE-ESY")، ومن ثم تخزينها وقراءتها من المخزن.

## كيف يعمل البرنامج؟
1.  يتم تهيئة المخزن وتصفير المؤشرات والعدادات.
2.  عند الكتابة، يتحرك مؤشر الرأس (`head`) للأمام مع كل حرف جديد.
3.  عند القراءة، يلحق مؤشر الذيل (`tail`) بمؤشر الرأس لسحب البيانات بالترتيب.
4.  في حال وصول أي مؤشر لنهاية المصفوفة، يعود تلقائياً للبداية (الموقع 0).
